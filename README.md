# MyDiary-MVVM
## Skills

- MVVM 패턴
- Realm CRUD
- Storyboard로 UI
- RxSwift

## 기능

- 일기 저장, 수정, 삭제 가능
- 일기 작성시, 자동으로 날짜와 날씨 불러옴
- 즐겨찾는 일기 찜 가능
- 비밀 일기 모드 선택시, 비밀번호 입력 후 열람 가능 (진행 중)
 

## 어려웠던 점

- RxSwift애 대한 이해와 적용
- MVVM패턴에 대한 적용과 폴더 구성
